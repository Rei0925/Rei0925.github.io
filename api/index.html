APIの参照方法(予定)

ServerChannelChatのAPIは、将来的に公開予定の JavaDoc や Dokka を参照して利用できます。

外部プラグインからチャネル操作やメッセージ送信を行う場合は、以下のような使い方を想定しています。

ServerChannelChatAPI.sendMessage(player, channelName, "メッセージ内容")

将来的な追加予定

  • 非同期チャネル操作のサポート
  • 権限付きチャンネル管理APIの拡張
  • チャネルイベントリスナーの追加
  • Dokkaによる自動生成ドキュメントの常時最新化

これらは将来的に順次実装される予定です。
